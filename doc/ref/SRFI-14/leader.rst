**************
SRFI-14 Module
**************

:lname:`Scheme`'s `SRFI-14`_ introduces the idea of a character set, a
*char-set*, whose membership can be tested.  These are used
predominantly in the regular expression matching code of `SRFI-115`_.

This module implements just enough to satisfy ``SRFI-115``.
